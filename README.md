<h1>Introduction</h1>
This project involves building a machine learning model to classify cases of monkeypox using a binary dataset. It employs advanced data preprocessing, feature extraction, and model evaluation techniques to achieve high accuracy in predictions. The project aims to assist in early detection and diagnosis, potentially aiding healthcare professionals in combating the spread of the disease.
<h1>Dataset overview</h1>
the dataset is uploaded on kaggle in the provided link (https://www.kaggle.com/datasets/nafin59/monkeypox-skin-lesion-dataset). the dataset is binary and it contains monkeypox and others.
